Welcome to SeamSmith

This site is a prototype site written from scratch by Matt Krabacher. It is meant to serve as a sample project and representation of skill and is not yet mobile ready. All code is availibe on GitHub and the site is hosted on an AWS EC2 Linux server. It was written primarily in TypeScript and JavaScript and was built with the following technologies and libraries.

  • Angular 5
  • Node.js
  • MongoDB
  • Mongoose
  • Express
  • Materialize
  • Sass/SCSS

Passwords are not encrypted and nothing is done with the email address. It is recommended that you use fake data for anything you put in. Nothing displayed on this website is actually for sale.
